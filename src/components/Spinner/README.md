Spinner types:

```jsx
<div>
  <Spinner type="round" />
  <br />
  <Spinner type="dotted" />
</div>
```

Spinners sizes:

```jsx
<div>
  <Spinner type="round" size="small" />
  <Spinner type="round" size="normal" />
  <Spinner type="round" size="large" />
  <br />
  <Spinner type="dotted" size="small" />
  <Spinner type="dotted" size="normal" />
  <Spinner type="dotted" size="large" />
</div>
```
